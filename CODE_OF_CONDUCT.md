
# Code of Conduct

## Our Commitment

We are committed to making ForgeCore a safe, inclusive, and respectful environment for all contributors.

We expect all participants to act professionally and constructively.

---

## Standards of Behavior

Examples of positive behavior:

- Respect differing opinions and approaches
- Accept constructive criticism gracefully
- Focus on what is best for the platform
- Be collaborative and helpful
- Prioritize clarity and technical quality

---

## Unacceptable Behavior

Unacceptable behavior includes:

- Harassment of any kind
- Disrespectful or offensive comments
- Personal attacks
- Disruptive or toxic communication
- Dismissing others without constructive reasoning

---

## Engineering Culture Expectations

Since ForgeCore is a backend infrastructure project, we also expect:

- Respect for architectural decisions
- Evidence-based technical arguments
- Focus on scalability and maintainability
- Avoiding ego-driven engineering debates
- Constructive review of code and design

---

## Enforcement

Project maintainers reserve the right to:

- Request behavior correction
- Reject contributions
- Remove contributors violating standards

---

## Scope

This Code of Conduct applies to:

- GitHub issues
- Pull requests
- Discussions
- All community interactions

---

Thank you for helping keep ForgeCore professional and welcoming.
